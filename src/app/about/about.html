Om denna laboration

I denna laboration har jag skapat ett CV med mina arbetserfarenheter. Jag har använt mig av MongdoDB som databasserver.

Funktionella krav

  • Webbplatsen använder Fetch API för att hämta (GET), skicka data (POST) samt radera (DELETE) från och till webbtjänsten.
  • Datan valideras innan POST-anrop och tydliga meddelanden skrivs ut vid valideringsfel.
  • Frontenddelen har skapats med HTML, CSS och JavaScript. Jag har i denna uppgift jobbat med Angular.

Slutsatser

Jag älskade Angular så mycket i laboration 2 i TypeScript kursen att jag gärna ville göra det här också. Insåg dock att mina kunskaper är väldigt bristfälliga vilket gjort att jag spenderat mycket tid på att framförallt lösa alla funktioner.
Det var kul att testa på en annan typ av databas men jag ska ärligt säga att det är för nytt allting för att jag ska kunna dra några egentliga slutsatser kring vad jag föredrar och hur allt skiljer sig.
